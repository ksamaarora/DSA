### Problem: 234. Palindrome Linked List

**Difficulty:** Easy

Given the `head` of a singly linked list, return `true` if the list is a palindrome, otherwise return `false`.

### Example 1:

**Input:** `head = [1, 2, 2, 1]`  
**Output:** `true`

### Example 2:

**Input:** `head = [1, 2]`  
**Output:** `false`

### Constraints:

- The number of nodes in the list is in the range `[1, 10^5]`.
- `0 <= Node.val <= 9`
